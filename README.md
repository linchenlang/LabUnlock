# LabUnlock - 机房教学管控软件终极清除工具




**一键清除机房三大监控/管理软件**：红蜘蛛、极域电子教室、学生机房管理助手。  
适用于学校机房、培训中心等被强制管控的电脑，解除束缚，恢复自由操作。

> **免责声明**：本工具仅用于学习研究或合法解除对自己电脑的不当管控。请勿用于破坏教学秩序或侵犯他人权益。使用本工具造成的一切后果由使用者自行承担。

---

## 功能特点

- **强制结束进程**：暴力结束红蜘蛛、极域、机房助手的进程及守护进程。
- **禁用驱动服务**：禁用极域的 `TDFileFilter`、`TDNetFilter` 等底层驱动。
- **映像劫持**：将目标程序重定向到 `systray.exe`，使其无法启动。
- **删除文件目录**：删除软件安装目录、驱动文件及临时生成物。
- **模糊匹配变种**：针对机房管理助手每天变换进程名的行为，自动匹配 `p*.exe` 并结束。

---

## 下载与使用

1. 从 [main](../main) 下载 `LabUnlock.bat`
2. **以管理员身份运行**（右键 → “以管理员身份运行”）
3. 阅读警告提示，按任意键开始清除
4. 等待脚本执行完毕

---

## 重要：通常需要运行 **两次**

由于极域电子教室等软件具有**进程守护**和**驱动级自我保护**机制，首次运行时可能仍有残余进程或服务未能完全卸载。  
**建议操作流程**：

1. 第一次运行脚本 → 等待完成
2. **观察右下角任务栏**：如果仍看到极域/红蜘蛛/机房助手的图标（或右键菜单中仍有管控软件），
3. **再次以管理员身份运行脚本**（无需重启电脑）
4. 第二次运行后，绝大多数情况下管控软件会被彻底清除

> **原理**：第一次运行结束后，部分守护进程被终止但可能被系统自动重启；第二次运行时，因为相关服务和驱动已被禁用或劫持，残留进程会被一网打尽。

---

## 验证是否清除成功

- 右下角任务栏无异常图标
- 按 `Ctrl+Alt+Delete` 打开任务管理器，检查是否有 `REDAgent.exe`、`StudentMain.exe`、`jfglzs.exe` 等进程
- 尝试运行极域学生端（如已被映像劫持，会无反应或报错）

---

## 恢复方法

如果电脑安装了**冰点还原**、**影子系统**等还原软件，重启即可恢复原状。  
否则，可通过以下方式手动恢复：

- 删除注册表中的映像劫持项（`HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\` 下对应程序名）
- 重新启用被禁用的服务（`sc config TDFileFilter start= demand`）
- 重装管控软件
